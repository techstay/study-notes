# Bash

Bash (Bourne Again Shell) is one of the most widely used Linux shells, coming pre-installed on virtually every Linux distribution. This universal availability makes Bash the de facto standard for writing portable shell scripts—most shell scripts you'll encounter in the wild are written in Bash. However, being designed decades ago, Bash shows its age in several areas. For simple automation tasks, Bash gets the job done. But for more complex requirements, a modern scripting language like Python offers better performance, easier development, and greater maintainability.

## Development Environment

When writing Bash scripts in VS Code, these extensions are highly recommended:

- **ShellCheck** — static analysis tool that detects errors, bugs, and suspicious patterns in your scripts
- **shell-format** — formats your script with proper indentation for better readability

## Learning Resources

- [TLDP Bash Beginner's Guide](https://tldp.org/LDP/Bash-Beginners-Guide/html/index.html) — comprehensive introductory tutorial, though somewhat dated
- [FreeCodeCamp Bash Scripting Tutorial](https://www.freecodecamp.org/news/bash-scripting-tutorial-linux-shell-script-and-command-line-for-beginners/) — practical tips and real-world examples for beginners
- [Bash Academy](https://guide.bash.academy/) — excellent visual explanations of shell syntax, though still a work in progress

# Bash（中文）

Bash（Bourne Again Shell）是 Linux 下最广泛使用的 Shell 之一，几乎所有 Linux 发行版都预装了 Bash。这种普遍性使 Bash 成为编写可移植 Shell 脚本的事实标准——你在实际项目中遇到的大多数脚本都是用 Bash 编写的。然而，Bash 由于诞生年代较早，在某些方面已经显现出局限性。对于简单的自动化任务，Bash 足够胜任。但对于更复杂的需求，现代脚本语言如 Python 提供了更好的性能、更便捷的开发体验和更强的可维护性。

## 开发环境

在 VS Code 中编写 Bash 脚本时，推荐安装以下扩展：

- **ShellCheck** — 静态分析工具，可检测脚本中的错误、缺陷和可疑写法
- **shell-format** — 自动格式化脚本，保持代码整洁易读

## 学习资源

- [TLDP Bash 新手指南](https://tldp.org/LDP/Bash-Beginners-Guide/html/index.html) — 内容全面的入门教程，但有些过时
- [FreeCodeCamp Bash 脚本教程](https://www.freecodecamp.org/news/bash-scripting-tutorial-linux-shell-script-and-command-line-for-beginners/) — 面向初学者的实用技巧和真实案例
- [Bash Academy](https://guide.bash.academy/) — 优秀的语法图解教程，仍在持续完善中
