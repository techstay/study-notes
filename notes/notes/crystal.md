Ruby-like static type language.

## Get started

Install on windows:

```sh
scoop install git
scoop bucket add crystal-preview https://github.com/neatorobito/scoop-crystal
scoop install vs_2022_cpp_build_tools crystal
```

Then install the vscode extension [crystal](https://marketplace.visualstudio.com/items?itemName=crystal-lang-tools.crystal-lang).
