### Installation

```sh
scoop install lua
```

### Editor

I use VSCode as my development environment and need to install the following extensions:

- [Lua Extension](https://github.com/sumneko/lua-language-server)
- [Lua Debug Extension](https://marketplace.visualstudio.com/items?itemName=actboy168.lua-debug)
