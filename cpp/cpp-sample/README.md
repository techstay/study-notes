# cpp-sample

C++ sample code collection, organized by topics.

## Directory Structure

- **[basic/](basic/)** - Basic syntax: variables, control flow, conditionals, loops, etc.
- **[function/](function/)** - Functions: function overloading, recursion, lambda, templates, namespaces, etc.
- **[oop/](oop/)** - Object-oriented: classes, inheritance, polymorphism, copy/move constructors, etc.
- **[stl/](stl/)** - Standard Library: string, vector and other container usage
- **[array/](array/)** - Arrays: raw arrays, std::array, iterators, etc.
- **[type/](type/)** - Type system: type conversion, constants, literals, etc.
- **[operator/](operator/)** - Operators: arithmetic, logical, bitwise operations, etc.
- **[modern/](modern/)** - Modern C++ features: modules, ranges, numeric, etc.
- **[io/](io/)** - Input/Output: cin, cout, file streams, string streams, etc.
- **[exception/](execption/)** - Exception handling: try/catch, rethrow, etc.
- **[boost/](boost/)** - Boost library examples
